Proyek ini merupakan simulasi interaktif rangkaian listrik seri dan paralel yang dibangun menggunakan HTML, CSS, dan JavaScript dengan fokus utama pada manipulasi Document Object Model (DOM) untuk mengubah atribut src pada elemen gambar lampu. Sistem ini mengimplementasikan dua jenis rangkaian, yaitu rangkaian tiga lampu dan empat lampu, yang dilengkapi dengan fungsi logika untuk menyalakan lampu secara berurutan menggunakan operator modulus, kontrol serentak untuk seluruh lampu, serta pengelompokan lampu spesifik (grup A,B dan C,D) menggunakan aset visual pic_bulboff.gif dan pic_bulbon.gif. Seluruh antarmuka didesain dengan tema gelap yang minimalis untuk memastikan pengalaman pengguna yang bersih dan fungsional sesuai dengan kriteria tugas praktikum pemrograman web.
